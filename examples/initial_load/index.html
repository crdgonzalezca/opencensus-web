OpenCensus Web Hello World!

This example demonstrates client side browser spans in the same trace as server side spans for an initial page load.

Because the browser does not send a trace context header for the initial page load, the server generates the parent trace context and sends it back in the HTML template in a <script> tag by setting the global traceparent variable according to the W3C Draft Trace Context standard.

The traceparent value received from server was "{{.Traceparent}}".
